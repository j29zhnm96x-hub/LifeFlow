# LifeFlow GTD Task Planner

## Project Documentation
LifeFlow is a minimalist Getting Things Done (GTD) Task Planner application that focuses on simplicity and ease of use.

### Setup Instructions
1. Clone the repository.
2. Open `index.html` in your web browser.
3. The application is ready to use.

### Features List
- Task management (create, read, update, delete)
- Voice input support
- Dark mode toggle
- Kanban board functionality
- Data export to JSON/CSV

### Usage Guide
Simply use the quick-add bar to create new tasks or manage existing ones through the sidebar navigation.

### Keyboard Shortcuts
- `Ctrl + N`: Create a new task
- `Ctrl + D`: Delete a task
- `Ctrl + S`: Save a task

### Data Storage Explanation
Tasks are stored in localStorage or IndexedDB for offline support.

### PWA Installation
You can install the app directly on your device for offline use.

### Browser Compatibility
LifeFlow is compatible with all modern browsers.

### Future Enhancements
- Integration with calendar apps
- Enhanced analytics for task management

### Architecture Overview
The app follows a Model-View-Controller (MVC) architecture, ensuring a clear separation of concerns.
